Implement Landing Page + About Section (Client Handoff UI)

We are preparing IndusAI for **client delivery**.

The current UI drops the user directly into internal tools (Chat / PLC Logs / Settings).
This is not appropriate for a delivered product.

We need a **clear entry point and simple explanation layer** for the system.

However, we **do NOT want a large documentation portal**.

Instead, we will implement:

• **Home Page (Landing Page)**
• **About Section (simple technical explanation pages)**

The UI must remain **clean, simple, and engineer-friendly**.

This is a **single-user industrial engineering tool**, not a SaaS documentation platform.

Add two new UI sections:

2. **About**

The goals are:

* Explain the project clearly
